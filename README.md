# hackathon-fiap-front-end

## Tecnologias utilizadas

- Node.js
  - NPM
  - Next.JS
- Tailwind CSS
  - Catalyst UI Kit

## Como executar

1. Primeiro, instale as dependências com o seguinte:

```bash
npm install
```

2. Depois, suba o servidor para a aplicação:

```bash
docker compose up
```

3. Por último, acesse [http://localhost:3000](http://localhost:3000) no seu navegador para visualizar o site.
